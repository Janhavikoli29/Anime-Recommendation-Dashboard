# Anime Analytics Dashboard | Power BI

## Project Overview

Anime has become one of the fastest-growing entertainment industries, with thousands of titles spanning multiple genres, formats, and audience preferences. Analyzing anime data can reveal valuable insights into ratings, popularity, episode counts, genres, and viewer engagement.

This project presents an interactive Anime Analytics Dashboard developed using Microsoft Power BI. By transforming raw anime data into meaningful visualizations, the dashboard enables users to explore anime ratings, genres, episode trends, member statistics, and content types through an intuitive and interactive interface.

## Project Objectives

* Analyze anime ratings across different genres and content types.
* Compare audience engagement using member statistics.
* Explore episode distribution among anime formats.
* Identify the most popular genres based on viewer participation.
* Provide an interactive dashboard for exploring anime trends.

## Dashboard Overview

The dashboard provides a comprehensive overview of anime data using KPI cards and interactive visualizations.

### Key Performance Indicators

* Average Rating
* Average Number of Members
* Average Number of Episodes
* Total Number of Members

### Dashboard Analysis

* Average Members by Genre
* Average Rating by Anime Type
* Average Members by Anime Type
* Average Episodes by Genre
* Average Rating by Genre
* Average Episodes by Anime Type

## Key Insights

* TV anime have the highest average number of members compared to other anime formats.
* Thriller and Psychological genres attract a larger average audience.
* TV series generally have significantly more episodes than movies, OVAs, or Specials.
* Most anime genres maintain average ratings between 6.0 and 7.5, indicating consistent audience reception.
* Different anime formats show clear differences in episode count and viewer engagement.
